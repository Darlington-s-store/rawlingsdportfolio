# Asomani Rawlings Junior - Software Engineer Portfolio

Personal portfolio showcasing full-stack development projects, technical skills, and professional experience.

## Tech Stack
- **Frontend**: React, Next.js, Tailwind CSS, Shadcn UI
- **Backend**: Node.js, Express.js, PostgreSQL, MySQL
- **Tools**: Vite, Firebase, Git

## Projects
- **Wedding Management**: Full-stack platform for event bookings and gift registries.
- **TripEase Ghana**: Transport and logistics management dashboard.
- **Drive & Thrive**: E-commerce marketplace for automotive accessories.
- **AtlasWave Travels**: Tour discovery and booking platform.
- **SparkLMS**: Interactive student learning management system.

## Setup & Development
1. Clone the repository:
   ```bash
   git clone https://github.com/Darlington-s-store/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run developmental server:
   ```bash
   npm run dev
   ```
